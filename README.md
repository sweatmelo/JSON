# Mongodb admin

- Next.js
- Mui
- Typescript

## Structure

```
pages # router
src
  - components
  - constants
  - hooks
  - libs
  - providers
  - utils
```

## How to dev

Clone this repo

```
cd mongodb-admin

yarn install

yarn dev
```

## How to deploy

```bash
yarn install

yarn build

yarn start

# port 8080
```
